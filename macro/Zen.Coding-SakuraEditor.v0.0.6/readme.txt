  概要
＝＝＝＝＝＝＝＝＝＝

  このソフトウェアはサクラエディタでZen Codingを利用するためのマクロです。

  Zen CodingはSergey Chikuyono氏によって作られ、
  公式サイトはGoogleCode(http://code.google.com/p/zen-coding/)です。



  ZenCodingとは
＝＝＝＝＝＝＝＝＝＝

  ZenCodingは"ul.items>li*3>a"といった文字列を

  <ul class="items">
    <li><a href=""></a></li>
    <li><a href=""></a></li>
    <li><a href=""></a></li>
  </ul>

  のように展開するもので、マークアップの効率を上げることができます。



  インストール
＝＝＝＝＝＝＝＝＝＝

  サクラエディタのヘルプ内"「登録済みマクロ」から実行"と同様です。
  
  Webでは下記のWikiにも記述があります。
  ・マクロファイルの実行
    http://sakura.qp.land.to/?Macro
  
  参考例を下記のページに掲載しておりますので参考にしていただければと思います。
  ・Zen Coding for SakuraEditor - MoonWing
    http://mwlab.net/zen-coding-for-sakuraeditor



  使用方法
＝＝＝＝＝＝＝＝＝＝

  Zen Codingの書式を入力し、マクロを実行します。

  書式については公式サイトを参照してください。

  ・Zen HTML Selectors
    http://code.google.com/p/zen-coding/wiki/ZenHTMLSelectorsEn

  ・Zen Coding cheat sheet (PDF)
    http://code.google.com/p/zen-coding/downloads/detail?name=ZenCodingCheatSheet.pdf
